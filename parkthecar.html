Park The Car

A simple to use responsive web application that helps user to find nearby parking lots as well as parking lots near predefined points of interest. Application created using HTML5, CSS3, Bootstrap, JQuery and Knockout.js following MVVM architecture. The information about parking lots and points of interest has been captured from Google Places API.
I also built a mobile application for Android phones.

To visit the live website, please click the link below. Remember to click on "Your Current Location" to find the parking spots around you!
